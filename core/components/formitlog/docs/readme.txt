--------------------
Extra: FormItLog
--------------------
Version: 0.0.1 (Revolution compatible)
Author: Kilian Bohnenblust (sofasurfer.org) 

A simple hook for FormIt to store form data in a jSon file.

Example Usage:
   [[!FormIt?hooks=`FormItLog,email,redirect`
   &logfile=`[[++formitlog.logfile]]`
